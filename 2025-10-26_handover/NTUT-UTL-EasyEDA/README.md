# 如何接手 EasyEDA 專案

## 適用對象

以下條件滿足其一者
1. 要使用 EasyEDA 從零開始開發電路的人
2. 要接手目前在 EasyEDA 的專案的人
3. 要接手 `NTUT-UTL-nRF-Connect-SDK-Monorepo` 的人 (因為部分專案與其相關)

以下條件皆滿足者
1. 稍微懂電路設計的人
2. 至少碰過一種電路設計軟件

## 開始

目前所有的專案都放在 `utlb07@gmail.com`，請使用 Google 帳號進行登入。  
若不知道帳戶密碼，可以去找教授或同儕詢問。
